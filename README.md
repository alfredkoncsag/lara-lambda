## Documentation

Lara-Lambda is a quick way to prepare a [Laravel](https://laravel.com), [Lumen](https://lumen.laravel.com) or [Laravel Zero](https://laravel-zero.com) project to run on [AWS Lambda](https://aws.amazon.com/lambda/).
